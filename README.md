# Quick About Info

## Why these Extensions exist
The main reason for this is that I use browser extensions on a daily basis, but I don't really have a trust relationship with the extension publishers, and some of the data that passes through extensions are sensitive in nature - so I should be able to trust them, shouldn't I?

Anyway, because of the slight doubt in trust, and the relatively frequent occurrence of browser extensions either being taken over, or simply bought out by malicious actors I thought the best course of action would be to create my own that I can have absolute trust in.

If anyone uses my extensions, and finds this page, know that I never intend to send any content that you pass through my browser extensions anywhere. The data may be stored locally, and may even be made exportable, but it will always be under the absolute control of the user and will never be sent to any endpoint on the internet (or elsewhere) with or without the user's knowledge.

## AI was used during the Creation of these Extensions
AI was used extensively in the creation of these extensions. But you don't have to trust that there are no mistakes, clone the repositories and satisfy yourself, no worries.

I am calling this out explicitly because some folk have moral objections to the use of AI. I acknowledge and even agree with those objections to an extent, however, that particular horse has bolted. Resisting the use of AI now is only holding you back, so I've decided to embrace it.

## What is the word Sheentee?
Sheentee is the phonetic spelling of the Irish work síntí, which means "extensions". I created this organisation to begin publishing browser extensions to assist with tasks useful for web application security testing.
